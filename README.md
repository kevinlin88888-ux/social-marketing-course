# 社群行銷經營術（FB／IG）

這是一個可直接部署到 Vercel 的靜態課程網頁。

## 檔案

- `index.html`：主網頁
- `assets/social-course-visual.png`：首頁視覺圖
- `vercel.json`：Vercel 靜態站設定

## Vercel 設定

如果從 GitHub 匯入此專案，請將 Vercel 的 Root Directory 設為：

```text
social-marketing-course
```

Framework Preset 可選：

```text
Other
```

Build Command 與 Output Directory 可留空。
